# JUnit 4 Sample Project

This is a sample Maven project using JUnit 4 for testing.

## Structure

- `src/main/java/com/example/Calculator.java` - Main calculator class
- `src/test/java/com/example/CalculatorTest.java` - JUnit 4 tests

## Build and Test

```bash
cd junit4
mvn clean test
```

## Package

```bash
mvn package
```

The JAR will be created in `target/junit4-sample-1.0.0.jar`
